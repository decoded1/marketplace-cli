## CLI Tool Options
 
### Status: No Dedicated CLI Tools Available
 
Currently, there are **no dedicated command-line tools** specifically for scrap metal pricing. However, you can use the APIs listed above with standard CLI tools:
 
#### Using curl with Metals-API:
```bash
curl "https://metals-api.com/api/latest?access_key=YOUR_KEY&base=USD&symbols=XCU,XAL"
```
 
#### Using curl with MetalpriceAPI:
```bash
curl "https://api.metalpriceapi.com/v1/latest?api_key=YOUR_KEY&base=USD&currencies=XCU,XAL"
```
 
#### Using curl with Metals.Dev:
```bash
curl "https://metals.dev/api/v1/latest?api_key=YOUR_KEY&currency=USD&metal=gold"
```
 
#### Building Custom CLI Tools:
The APIs above can be integrated into custom CLI tools using:
- Python with `requests` library
- Node.js with `axios` or `node-fetch`
- Go with `net/http` package
- Shell scripts with `curl` and `jq` for JSON parsing

## Important Considerations
 
### Data Accuracy:
- User-reported prices (iScrap) may vary from actual yard prices
- Always call ahead to confirm rates before traveling
- Prices fluctuate based on commodity markets, location, and volume
 
### Legal/Compliance:
- Some regions require ID for scrap metal sales
- Certain items (catalytic converters, wire) have additional regulations
- Keep receipts for tax purposes
 
### Best Practices:
- Check multiple sources before selling large quantities
- Monitor trends for several days/weeks for timing
- Build relationships with local yards for better rates
- Clean and sort metals for higher payouts
- Understand metal grades and classifications
